# IT Ticketing System

A web-based IT help desk ticketing system built using Python, Flask, and SQLite.

## Features

* Create support tickets
* Assign tickets to technicians
* Set ticket priority levels
* Store tickets in a SQLite database
* View all submitted tickets

## Technologies Used

* Python
* Flask
* SQLite

## How to Run

1. Install Flask:

pip install flask

2. Run the application:

python main.py

3. Open your browser and visit:

http://127.0.0.1:5000
